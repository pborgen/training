# Training iOS - TestFlight Checklist

- [ ] Install XcodeGen (`brew install xcodegen`)
- [ ] Generate project (`xcodegen generate`)
- [ ] Add `GoogleService-Info.plist` to app target
- [ ] Add URL Type (reversed client id)
- [ ] Set Apple Team + Signing
- [ ] Add app icon assets
- [ ] Archive from Xcode (Any iOS Device)
- [ ] Upload to App Store Connect
- [ ] Add internal testers in TestFlight
